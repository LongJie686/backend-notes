## 实战项目

本站笔记对应两个开源实战项目，将笔记中的理论知识落地为可运行的完整系统。

### Synapse -- 多智能体 AI 应用平台

**GitHub**: [LongJie686/synapse](https://github.com/LongJie686/synapse)

基于 Python + FastAPI 构建的 Multi-Agent AI 应用平台，涵盖本站 **AI 应用** 领域全部知识点。

| 笔记 | 项目实现 |
|------|---------|
| 多智能体设计实战 | Supervisor / Parallel / Hierarchical / Collaboration 四种编排模式 |
| 大模型应用开发 | LLM Router + Prompt 模板 + RAG Pipeline + Function Calling |
| Agent 架构分析 | Plan-Execute 编排、RAG + 工具链、可观测性追踪 |
| MCP 协议 | MCP Client 集成 |
| Skills 技能系统 | 动态技能加载 + 自动演化 |
| 安全护栏 | 注入防御 + 内容审核 + 成本控制 |

### Ecommerce Microservices -- 电商推荐系统微服务架构

**GitHub**: [LongJie686/ecommerce-microservices](https://github.com/LongJie686/ecommerce-microservices)

基于毕设项目重构的 Python + FastAPI 微服务架构，涵盖本站 **MySQL / 高并发 / 架构 / 微服务** 全部知识点。

| 笔记 | 项目实现 |
|------|---------|
| MySQL 学习笔记 | 表设计 + 复合索引 + 覆盖索引 + 逻辑删除 + 事务 + 读写分离 + EXPLAIN |
| 高并发系统设计 | 多级缓存 + 防穿透/雪崩/击穿 + 分布式锁 + 熔断降级 + 限流 |
| 架构设计 | 三层架构 + 策略模式 + 统一响应 + 配置管理 |
| 微服务 | 6 服务独立部署 + API 网关 + Kafka 异步通信 + Docker Compose |
